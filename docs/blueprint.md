# **App Name**: Annadata AI

## Core Features:

- Location-Based Crop Suggestion: Automatically detect the farmer's location using GPS to determine the best crops to grow based on climate, soil, and season. Includes a manual location selection fallback. Leverages weather APIs and historical data to inform crop selection.
- Smart Crop Recommendation Engine: Suggests suitable crops with detailed information like local and English names, high-quality images, suitable soil type, ideal temperature and rainfall, expected yield per acre, and estimated market price.
- Crop Life-cycle Guide: Provides a complete guide for each recommended crop, including the best sowing time, seed rate, step-by-step fertilizer schedule, medicine/pesticide guide, irrigation schedule, and best harvesting time.
- Multi-Language Support: Supports multiple Indian local languages (Hindi, English, Marathi, Tamil, Telugu, Bengali) with a simple toggle to change the language. Uses farmer-friendly, easy-to-understand wording. Leverages Google Translate API for smart translation.
- Weather-Based Alerts: Sends weather-based alerts (rain, heatwave, frost) to help farmers take timely action and mitigate risks. An admin-defined risk assessment model determines which risks apply.
- "Aaj Ka Sujhav" - Daily Farming Tip: Provides daily farming tips. The tips are curated by the admin and displayed based on the current weather conditions.
- Voice Read-Out: Enables voice read-out of crop information, farming tips, and government schemes in the selected local language. Text-to-speech uses the translated text obtained using the Google Translate API.

## Style Guidelines:

- Primary color: Forest Green (#1B5E20), representing growth and nature. This should provide enough contrast on the background color
- Secondary color: Very light desaturated green (#F1F8E9), offering a clean and calming backdrop.
- Accent color: Amber (#FFB300), used for highlighting key information and calls to action, complementing the green tones.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a blend of modernity and warmth for easy readability.
- Use clear, representative icons from Lucide/Hero Icons for crops, weather conditions, and farming activities.
- Mobile-first, clean layout with large buttons and readable fonts optimized for low-end Android phones, tablets, and desktops.
- Smooth transitions and subtle animations to enhance user experience without being distracting.
